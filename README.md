# movie-site-info-collection-notes-v3
本仓库用于收集并整理影视类网站的公开信息，对公开网络中的电影网站进行基础资料归档。整理内容包括站点名称、访问地址、站点定位说明及基础分类结构，便于长期留存与信息比对。本项目不涉及影视资源的解析、播放或下载，仅对公开信息进行结构化整理。所有数据均来自公开渠道，仅供资料查询与研究参考使用，不代表对任何第三方站点的认可或评价。
